CampusFusion is a web-based food ordering system designed specifically for college and campus environments.  
It helps students quickly browse available food items, place orders, track order status, and raise complaints if needed.  
The admin panel allows stall owners / canteen managers to efficiently manage menu items, orders, and user requests.



## Features

### For Students / Users
- Register & Login to the system
- Browse menu items with images and price
- Add quantity and place food orders
- View past order history
- Raise support tickets for queries or complaints
- Receive updates on order status in real-time

### For Admin (Canteen / Stall Owner)
- Login to Admin Dashboard
- Add, edit, or delete food items from the menu
- View & manage all incoming orders
- Update order status (Pending, Cooking, Completed, Delivered, Cancelled)
- View and reply to tickets raised by users
- Manage user accounts if required
